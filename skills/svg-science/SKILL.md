# SVG Science

Generate publication-style SVG assets with flat academic styling, SimSun-first fonts,
clear spacing, and non-overlapping layout primitives.

## Triggers

- `svg`
- `scientific figure`
- `workflow diagram`
- `framework diagram`
- `timeline figure`
- `academic illustration`

## Commands

- `workflow`
- `framework`
- `timeline`
